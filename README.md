Getting Started
============

### First install all the necessary dependencies.
```
 yarn install
 npm install
```

### Start the development server (changes will now update live in browser)
```
yarn start
npm start
```
To view your project, go to: http://localhost:3000/


### Build to Production
```
yarn build
npm build
```
